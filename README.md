Markdown Server
====

### 说明
执行命令在当前目录启动一个 markdown 服务器

### 使用方法
```
markdown-server
```

### 关于主题
非常感谢 [MarkdownCSS] 提供的 markdown 样式表文件，
以及 [github-markdown-css] 提供的适用于 Github 的 markdown 样式

### 已知 Bugs
- 静态资源目录的对路径的占用
